# HSK4 単語学習アプリ

HSK4級（中国語検定）合格のための単語学習Webアプリです。全1200語を収録し、ピンイン・漢字・日本語訳・例文の表示、発音再生機能を備えています。

## 機能

- **フラッシュカード学習** - 漢字・ピンイン・意味・例文を1枚ずつ表示
- **発音再生** - Web Speech APIによる中国語音声読み上げ（速度調整可能）
- **カテゴリフィルター** - 名詞・動詞・形容詞・副詞・その他で絞り込み
- **検索** - 漢字・ピンイン・日本語訳からの単語検索
- **単語一覧** - テーブル形式での一覧表示
- **簡易クイズ** - 漢字→意味／意味→漢字の4択クイズ
- **学習進捗管理** - 学習済み・お気に入りをlocalStorageに保存

## 技術スタック

- React 18 + TypeScript
- Vite
- CSS Modules
- React Router v6
- Web Speech API

## 使い方

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動（localhost:5173）
npm run dev

# 本番用ビルド
npm run build

# ビルド結果のプレビュー
npm run preview
```

## スクリーンショット

```
/
├── メイン画面（フラッシュカード）
├── /list（単語一覧）
├── /quiz（クイズモード）
└── /settings（設定）
```

## データ

HSK4級公式単語リストに基づく全1200語を収録（`src/data/words.json`）。
